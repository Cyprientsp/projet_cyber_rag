# HX-Solar 5000 — Fiche produit technique (fictif)

## Présentation
L'onduleur HX-Solar 5000 est le produit phare de HelioniX SAS. Il transforme le
courant continu des panneaux solaires en courant alternatif pour la maison.

## Caractéristiques techniques
- Puissance nominale : 5 000 W (5 kW)
- Tension d'entrée MPPT : 120 V à 500 V
- Nombre de trackers MPPT : 2
- Rendement européen : 97,8 %
- Rendement maximal : 98,4 %
- Protection : IP65 (utilisable en extérieur)
- Poids : 12,3 kg
- Température de fonctionnement : -25 °C à +60 °C
- Garantie constructeur : 12 ans

## Connectivité
Le HX-Solar 5000 intègre le module de supervision **HelioCloud**, accessible via
l'application mobile HelioniX. Le protocole de communication interne se nomme
**SunLink v3** et fonctionne sur le port TCP 8770.

## Prix public conseillé
Le prix public conseillé est de 1 690 € HT (hors pose), en avril 2025.

## Référence SAV
Le code d'erreur **E-204** signale une surchauffe du transformateur : il faut
vérifier la ventilation et contacter le SAV HelioniX si le défaut persiste.
